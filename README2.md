# Proyecto02
     
Instrucciones para correr el programa:
Primero debemos instalar en la terminal:
pip install opencv-python
pip install numpy
pip install pillow

  ```pip install opencv-python```
  
  ```pip install numpy```
  
  ```pip install pillow```

Una vez que tengamos instaladas las librerías podemos continuar con el programa, se encontrará una carpeta llamada Proyecto02 en el que se encontrarán todos los archivos que necesitamos para que este funcione. 
PARA DESCARGAR EL PROGRAMA, se puede hace de las siguientes formas:

* Escribir en la terminal 


         git clone https://github.com/Brendaaf2003/Proyecto02.git
    
    
* Descargar el archivo en forma de zip

Después se debera agregará la imagen que se desea consultar en la carpeta, entonces para ver su clasificación además de su color buscaremos el archivo que se llame “main”, en el que se deberá escribir la siguiente línea en la terminal 

**Para Linux o Windows:**

     python main.py

**Para MacOs:**
  
     python3 main.py
     
     
Este archivo te pedirá que escribas el nombre de la imagen con terminación **.bmp**, por lo tanto el programa procederá a leer la imagen para que aparezca en pantalla lo que se requiere, cuando imprima la categoría se tiene que presionar enter para que se pueda ver el valor hexadecimal entonces automáticamente aparecerá un mensaje en el que preguntará si deseas continuar con otra figura, en caso de que así lo desee se deberá escribir un 1 y en caso contrario cualquier otra letra.
Por lo tanto si se desea agregar más imágenes, entonces cada que termine el programa de nuevo deberá escribir 

**Para Linux o Windows:**

    python main.py

**Para MacOs:**

    python3 main.py

Pero ahora escribirá el nuevo nombre de la imagen.
